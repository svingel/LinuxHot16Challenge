# 1. sudo
# Pozwala czuć się władcą absolutnym swojego systemu.

# Każde linuxowe dziecko kiedyś uczy się chodzić :D cd pozwala nam buszować po katalogach.

# 3. ls
# Kiedy już umiemy chodzić, warto jest też widzieć dokąd zmierzamy.

# 4. pwd
# kiedy umiemy już chodzić (cd) i opanowaliśmy zmysł wzroku (ls) warto wiedzieć gdzie jesteśmy, mapę wyjmujemy wpisując pwd

# 11. mv
